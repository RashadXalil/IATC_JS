includes adlı funksiya yazın. Funksiya bir ədəd array və bir ədəd axtarış üçün dəyər qəbul etməlidir. Əgər həmin dəyər array in içərisində mövcuddursa, true əks halda false qaytarmalıdır:

includes([5, 4, 3, 2, 1], 4); // cavab: true
includes([-1, 4, 3, 7, 9], 10) // cavab: false
includes(['a', 'b', 'c', 'd'], 'b'); // cavab: true
includes([], 13); // cavab: false includes(); // cavab: false
